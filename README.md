# FritzDect
PythonScript for FritzDect socket-outlets

#Config
The FritzDect module expects a config file.
The config file is a JSON file of this form:
'''
{
    "username": "user",
    "password": "pw",
    "url": "ip"
}
'''
You need to replace the *user*, *pw* and *ip* part to fit your application
